A simple, web-based task management application built with **HTML, CSS, and JavaScript**.  
This project allows users to add, edit, delete, and filter tasks with persistence using **Local Storage**.

## ğŸ“Œ Features

- **HTML Structure (5 points)**  
  - Clean and organized layout with input form and dynamic task list.  
  - Each task includes: **Title, Description, Due Date, Priority (Low/Medium/High)**.

- **Add Tasks (5 points)**  
  - `addTask()` function collects user input.  
  - Validates required fields (Title & Due Date).  
  - Dynamically adds tasks to the list.

- **Display Tasks (5 points)**  
  - `displayTasks()` function shows tasks dynamically.  
  - Displays all task details clearly.

- **Local Storage (10 points)**  
  - Saves tasks in browser storage.  
  - Loads tasks automatically on page refresh.  
  - Updates storage when tasks are added/edited/deleted.

- **Edit Tasks (5 points)**  
  - `editTask()` function allows modification of existing tasks.  
  - Each task has a unique identifier for accurate editing.

- **Delete Tasks (5 points)**  
  - `deleteTask()` function removes tasks.  
  - Updates UI dynamically.

- **Priority-based Filter (10 points)**  
  - Filter tasks by priority (Low/Medium/High).  
  - Search functionality for quick task lookup.

- **User Interaction (5 points)**  
  - Input fields clear after adding/editing tasks.  
  - Intuitive and user-friendly interface.
